# Vihome — AI-Powered Eco-Living Marketplace

A three-sided AI platform connecting eco-service providers (organic farmers,
solar installers, water harvesting), property owners, and buyers — so users
can find sustainable homes with all living resources bundled in one search.

## Tech Stack
- Frontend: React + Vite, Tailwind CSS
- Backend: FastAPI (Python)
- Database: Supabase (PostgreSQL + pgvector)
- AI: Anthropic Claude API (claude-sonnet-4-20250514)
- Auth: Supabase Auth
- Deployment: Vercel (frontend) + Render (backend)
